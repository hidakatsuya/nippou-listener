# NIPPOU Player

A client for playing NIPPOU in [esa.io](https://esa.io).

## Development

```bash
$ npm i

# Serve with hot reload at localhost:9080
$ npm run dev

# Build electron app for production
$ npm run build

# Lint all JS/Vue component files in `app/src`
$ npm run lint
```
